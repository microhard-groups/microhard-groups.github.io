# Groups

**A simple, modern web-based messaging app** — built for quick group chats and real-time conversations.

![Groups Preview](https://group-chats.github.io/favicon.png) <!-- Replace with a better screenshot if you add one -->

## ✨ Features

- **Real-time messaging** — Send and receive messages instantly
- **Group chats** — Create and join group conversations
- **Direct messages** — Private 1-on-1 chats
- **User authentication** — Sign up / Sign in
- **Customizable profile** — Status, theme color, wallpaper, and more
- **Settings & Privacy** — Online status, notifications, read receipts, etc.
- **Mobile-friendly** — Works great on desktop and mobile browsers
- **No backend required** (for now) — Pure frontend demo / local-first experience

Live demo: **[group-chats.github.io](https://group-chats.github.io/)**

## 🚀 Quick Start

1. Visit **[https://group-chats.github.io/](https://group-chats.github.io/)**
2. Sign up or sign in
3. Create a group or start chatting!
